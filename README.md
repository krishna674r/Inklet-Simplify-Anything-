## 🚀 Quick Start

### Prerequisites

- Node.js
- npm or yarn
- A **Gemini API Key** from [Google's Generative AI](https://ai.google.dev)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/krishna674r/inklet-Paste-anything-confusing.git
   cd inklet-Paste-anything-confusing
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure your Gemini API Key**
   - Open `.env.local` file
   - Add your Gemini API key:
     ```env
     GEMINI_API_KEY=your_api_key_here
     ```
   - Get your free API key from [Google's Generative AI](https://ai.google.dev)

4. **Run the application**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   - The application will be available at the address shown in your terminal

## 📖 How to Use

1. **Paste your content** - Copy and paste any confusing text into the input field
2. **Click Simplify** - Hit the simplify button and let AI work its magic
3. **Get your explanation** - Receive a clear, easy-to-understand breakdown
4. **Copy & Share** - Copy the simplified version or share it with others

## 🔑 Environment Variables

Create a `.env.local` file in the root directory:

```env
# Google Gemini API Key
GEMINI_API_KEY=your_gemini_api_key_here
```

## 🔐 Security

- API keys are stored securely in `.env.local`
- `.env.local` is added to `.gitignore` to prevent accidental exposure
- Never commit sensitive credentials to version control
- Use environment variables for all sensitive data
